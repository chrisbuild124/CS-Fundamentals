# Operations

- AND <br/>
n = 1 & 1

- OR <br/>
n = 1 | 0

- XOR <br/>
n = 0 ^ 1

- NOT (negation) <br/>
n = ~n

- Bit shifting <br/>
n = 1
n = n << 1
n = n >> 1

# General

- Binary bit representation <br/>
0xFFFFFFFF = 0b11111111111111111111111111111111

- Binary bit representation <br/>
0x7FFFFFFF = 0b01111111111111111111111111111111

- Convert an overflowed integer to signed <br/>
This will flip res bits by 1 and then take the inverse: res = ~(res ^ mask)
